# AirClient

A free open-source mixin-based injection hacked client for Minecraft using Minecraft Forge.

**Version:** b1.0 | **Minecraft:** 1.8.9

## English

### Introduction

AirClient is a free and open-source Minecraft hacked client built on the LiquidBounce legacy codebase. This project is **fully written by AI with skidded code**, featuring modern design and rich customization options.

### Highlights

- **200+ Theme Colors** - A complete theme color system with over 200 preset colors to personalize your client
- **Music Player** - A fully-featured music player with playlist support
- **Dynamic Island** - Beautiful dynamic island with lyrics display
- **Modern HUD** - Modern and customizable HUD interface
- **Personalization** - Rich customization options for every aspect of the client

### Modules

- **Combat** - KillAura, Velocity, Criticals, Aimbot, etc.
- **Movement** - Speed, Fly, Sprint, Step, etc.
- **Render** - ESP, Tracers, Chams, FullBright, etc.
- **Player** - AutoArmor, AutoTool, AutoEat, AutoClicker, etc.
- **World** - XRay, Nuker, ChestStealer, Scaffold, etc.
- **Misc** - Music Player, Theme Manager, HUD Editor, etc.

### Installation

1. Install Minecraft Forge for 1.8.9
2. Download the latest release from [Releases](https://github.com/lmx0721/AirClient/releases)
3. Put the jar file into your `.minecraft/mods` folder
4. Launch Minecraft with Forge profile

### Building

```bash
gradlew setupDevWorkspace build
```

## 中文

### 简介

AirClient 是一个免费开源的 Minecraft 作弊客户端，基于 LiquidBounce legacy 代码库构建。本项目**由 AI 全程编写 + skid 代码**，拥有现代化的设计和丰富的自定义选项。

### 特色功能

- **200+ 主题颜色** - 完善的主题颜色系统，提供 200+ 种预设颜色，打造个性化客户端
- **音乐播放器** - 功能完善的音乐播放器，支持播放列表
- **灵动岛** - 好看的灵动岛设计，支持歌词显示
- **现代化 HUD** - 现代化、个性化的 HUD 界面
- **高度可定制** - 丰富的自定义选项，满足各种需求

### 模块列表

- **战斗** - KillAura, Velocity, Criticals, Aimbot 等
- **移动** - Speed, Fly, Sprint, Step 等
- **渲染** - ESP, Tracers, Chams, FullBright 等
- **玩家** - AutoArmor, AutoTool, AutoEat, AutoClicker 等
- **世界** - XRay, Nuker, ChestStealer, Scaffold 等
- **杂项** - 音乐播放器, 主题管理器, HUD 编辑器 等

### 安装

1. 安装 Minecraft Forge 1.8.9
2. 从 [Releases](https://github.com/lmx0721/AirClient/releases) 下载最新版本
3. 将 jar 文件放入 `.minecraft/mods` 文件夹
4. 使用 Forge 配置文件启动 Minecraft

### 构建
