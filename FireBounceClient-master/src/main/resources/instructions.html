How to install FireBounce

  1. Download Forge for Minecraft 1.8.9 from minecraftforge.net.
  2. Install Forge.
  3. Open the Minecraft folder.
  4. Create a folder called mods.
  5. Copy this file into it.
  6. Run Minecraft with the Forge profile selected.
