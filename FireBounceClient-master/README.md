A powerful Minecraft Forge utility client designed for competitive gameplay, automation, and enhanced controls. Based LiquidBounce, but with improved performance and unique features.

✨ Features
🛠 Core Functionality
✔ Optimized Performance – Lightweight & lag-resistant
✔ Modular System – Toggle modules on/off dynamically
✔ Customizable GUI – Clean ClickGUI (Press RIGHT_SHIFT)
✔ Config System – Save & load multiple profiles

⚔ Combat Modules
KillAura – Auto-attack nearby enemies

MoreKB - Increases the knockback you deal to enemies

Velocity – Reduce or reverse knockback from your enemy

AutoClicker – Just keep pressing the left/right button to get a high CPS

🏃 Movement Modules
Flight – Vanilla, Matrix, Intave modes

Speed – Make you get more move speed

Spider – Climb walls like a spider

NoFall – Cancel fall Damage

NoSlow - Make you avoid slowness from using item or blocking

👀 Visual Modules
ESP – Highlight players/mobs

Tracers – Show direction to entities

FullBright – Permanent night vision

X-Ray – See ores through blocks

🧰 Utility Modules
AutoTool – Best tool selection

ChestStealer – Loot chests quickly

Scaffold – Auto-place blocks for bridging

InventoryCleaner – Drop useless items

📥 Installation
Requires:

Minecraft 1.8.9

Forge (latest recommended)

Steps:

Download FireBounce.jar from Releases or download the source code to build the jar yourself

Drop it into your mods folder

Launch Minecraft with Forge profile
